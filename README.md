# MagnetControl

 Página web interactiva y educativa diseñada para gestionar, registrar y visualizar experimentos relacionados con el magnetismo. La plataforma permite consultar información, realizar simulaciones del comportamiento de los polos magnéticos, registrar experimentos y visualizar indicadores mediante una interfaz moderna, intuitiva y responsive.
Tema seleccionado: Magnetismo.

Herramientas utilizadas: HTML5, CSS3, JavaScript, GitHub y Render.

Funcionalidad principal: Facilitar el aprendizaje y la gestión de experimentos de magnetismo mediante un dashboard interactivo, registro de experimentos, consulta de información, simulador del campo magnético, cuestionario educativo y herramientas de visualización de datos.
